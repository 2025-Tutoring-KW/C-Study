#### 1. C언어는 컴파일러이다. (==인터프리터가 아님.)

## 문자열
#### char 타입으로 되어있는 배열.
#### 문자열은 일반 정수형 배열과 다르게 마지막 자리에 \0(=NULL)값이 들어가야함.
#### 일반 변수 배열과 다르게 문자열 끝에 NULL값이 들어가야 하는 이유는 문자열은 strcpy, strcat으로 문자열의 크기가 자유자제로 변할 수 있음.(==가변적이다.)
#### 따라서 문자열의 끝을 구분하기 위해 문자열에 NULL값을 넣어줘야한다.
#### 문자 변수 1개를 담을 때는 ''로 묶고 문자열을 담을 때는 ""로 묶는다. 
#### ex) char a = 'C'; char str[6]="hello";
### 예제 코드
```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main() {
	int numArr[10] = { 11, 22, 33, 44, 55, 66, 77, 88, 99, 110 };

	char ch;

	char str[11];

	scanf("%c", &ch);

	scanf("%s", str);

	printf("%c %s", ch, str);

	return 0;
}
```

## 조건문
### if문
```c
if(조건문){
	조건문을 만족할때 실행할 코드;
}
```
### else if문
```c
if(조건문){
	if 조건문이 참일 때 실행할 코드;
}
else if(조건문){
	자신 위의 조건문들이 모두 거짓이며, else if 조건문이 참일 때 실행할 코드;
}
else if(조건문){
	자신 위의 조건문들이 모두 거짓이며, else if 조건문이 참일 때 실행할 코드;
}
else{
	if 조건문과 else if 조건문 모두 거짓일 경우 실행할 코드;
}
```
단 if를 else if 혹은 else와 같이 썼을 때 if ,else if 조건문에서 참이라면 그 뒤에 조건문을 확인하지 않고 다음 코드로 넘어감.

### swich문문
```c
int grade;
scanf("%d", &grade);
switch (grade) {
case 4:
    printf("훌륭해요!");
    break;
case 3:
    printf("좋아요!");
    break;
case 2:
    printf("평균이네요!");
    break;
case 1:
    printf("좋지 않아요!");
    break;
case 0:
    printf("좀 더 노력하세요!");
    break;
default:
    printf("잘못된 학점입니다");
    break;
}
```
 **swich문은 case마다 break;를 꼭 추가해야함. 추가하지 않을 시 case에 만족하는 조건문이 있을 때 아래에 있는 코드들을 모두 실행하며 넘어감.**

# 실습 예제
숫자 맞추기 게임

프로그램은 1 ~ 100까지의 임의의 수를 발생시킨다.  
사용자는 1~100사이의 값을 반복적으로 입력해서 컴퓨터가 생각한 값을 맞추면 게임이 끝난다.  
사용자가 값을 입력하면 컴퓨터는 앞에서 생성한 값과 비교해서 결과를 알려준다.  
사용자가 컴퓨터가 생각한 숫자를 맞추면 게임이 끝나고 몇 번 만에 숫자를 맞췄는지 알려준다. 
```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
	srand(time(NULL));
	int number = rand()%100 + 1;
	/* Write Your Code */

	return 0;
}
```
